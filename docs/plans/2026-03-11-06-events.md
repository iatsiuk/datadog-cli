# Events: search, list, get, create

## Overview

Implement the `events` command group for datadog-cli, providing CLI access to
Datadog Events APIs. Covers listing, searching, creating, and viewing events.

Uses `github.com/DataDog/datadog-api-client-go/v2` (datadogV2.EventsApi).

Depends on: foundation plan.

## Context

- SDK APIs used:

## Technical Details

### CLI structure

```
dd events list --query "source:github" --from "now-24h" --to "now" --limit 50
dd events search --query "deployment" --from "now-7d"
dd events show <event-id>
dd events create --title "Deploy v2.1" --text "Released to prod" --tags "env:prod,service:web" --alert-type info
dd events tail --query "source:pagerduty"
```

### List output columns

Table mode: `TIMESTAMP | TITLE | SOURCE | TAGS`

### Show output fields

Detail mode: ID, Title, Text, Date, Source, Tags, Alert Type, URL
