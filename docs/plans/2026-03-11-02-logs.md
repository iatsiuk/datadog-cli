# Logs: search, indexes, pipelines, archives, metrics, custom destinations, restriction queries

## Overview

Implement the `logs` command group for datadog-cli, providing CLI access to all
Datadog Logs APIs. Covers searching/listing logs, managing indexes, pipelines,
archives, log-based metrics, custom destinations, and restriction queries.

Uses `github.com/DataDog/datadog-api-client-go/v2` (datadogV2 for search/aggregate/archives/metrics/destinations/restrictions, datadogV1 for indexes/pipelines).

Depends on: foundation plan (config, client, output, root command).

## Context

- SDK APIs used:
  - `datadogV2.LogsApi`: ListLogs, ListLogsGet, AggregateLogs, SubmitLog
  - `datadogV1.LogsIndexesApi`: CRUD indexes, order
  - `datadogV1.LogsPipelinesApi`: CRUD pipelines, order
  - `datadogV2.LogsArchivesApi`: CRUD archives, roles, order
  - `datadogV2.LogsCustomDestinationsApi`: CRUD custom destinations
  - `datadogV2.LogsMetricsApi`: CRUD log-based metrics
  - `datadogV2.LogsRestrictionQueriesApi`: CRUD restriction queries, roles

## Technical Details

### CLI structure

```
dd logs search --query "service:web" --from "now-1h" --to "now" --limit 100
dd logs tail --query "service:web env:prod"
dd logs aggregate --query "service:web" --compute "count" --group-by "status"
dd logs index list
dd logs index show <name>
dd logs index create --name <name> --filter <query> --retention <days>
dd logs index update <name> [flags]
dd logs index delete <name> --yes
dd logs pipeline list|show|create|update|delete
dd logs archive list|show|create|update|delete
dd logs metric list|show|create|update|delete
dd logs custom-destination list|show|create|update|delete
dd logs restriction-query list|show|create|update|delete
```

### Search output columns

Table mode: `TIMESTAMP | SERVICE | HOST | STATUS | MESSAGE`
